# Mini Search Engine & Website Creator Welcome to the Mini Search Engine & Website Creator! 🚀 This project is a simple demonstration of how you can build your own mini search engine and website builder using Python and C++. Whether you want to search through your own collection of static websites or create and host your own pages, this project is the perfect starting point. **Features:** - 🔍 Search websites by keywords - 📝 Create your own simple HTML websites - 🖥️ Use either a web interface (Python Flask) or a simple console interface (Python or C++) ## Getting Started See the instructions and code files in this repository to learn how to run the web or console versions. Feel free to fork, modify, and expand! --- Get started, explore, and unleash your creativity!
